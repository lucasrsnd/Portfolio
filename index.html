Meus projetos

Aqui estão projetos que desenvolvi durante meus estudos.

Fit-Club

Desenvolvi uma página para uma academia, usando HTML e CSS para a estrutura e estilo. A tornai responsiva e funcional com JavaScript, incluindo validações de formulários e um carrossel.

Car-Spotting

Galeria de imagens utilizando JSON, HTML, Sass e JavaScript. Quando o usuário clica em uma das fotos na página inicial, é redirecionado para outra com uma geração aleatória de imagens do JSON.

Cafeterias-Bh

Criei uma landing page estática usando Bootstrap para um projeto da faculdade. A página tem a função de apresentar cafeterias na região de Belo Horizonte e seus principais produtos.
